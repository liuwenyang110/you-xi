# 游戏项目合集

这个仓库用于保存本地游戏相关项目与练习代码，包含 Python 小游戏、象棋项目，以及部分前后端综合练习项目。

## 主要内容

- `cursor代码/`：Python 游戏练习代码
- `游戏项目/`：中国象棋相关页面和 Vite 项目
- `新建文件夹/`：前后端综合项目代码
- `sql/`：数据库脚本目录

## 运行提示

不同子项目的运行方式不同，请进入对应子目录查看 `package.json`、启动脚本或源代码入口。

## 安全说明

已忽略 `.env`、真实 API Key、数据库文件、依赖目录、缓存、日志和大型软件运行目录。真实高德 API Key 没有提交到仓库。
