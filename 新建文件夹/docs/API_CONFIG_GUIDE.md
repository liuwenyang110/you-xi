## #1 #2 高德地图 Key 申请教程

两个 Key 在同一个高德账号、同一个应用下创建，只是"服务平台"选不同的。

### 第一步：注册高德开放平台

1. 打开 https://lbs.amap.com/
2. 右上角「注册」→ 手机号注册
3. 登录后点右上角头像 → 「认证」→ 选「个人开发者」
4. 填写姓名 + 上传身份证正面照 → 提交（秒审或几分钟）

### 第二步：创建应用

1. 进入控制台：https://console.amap.com/dev/key/app
2. 点「创建新应用」
3. 应用名称：`农助手`
4. 应用类型：`出行`或 `其他`

### 第三步：添加两个 Key

在刚创建的「农助手」应用下，点击「添加 Key」，分两次创建：

| Key 名称 | 服务平台 | 用在哪 |
|---------|---------|-------|
| `农助手后端` | **Web服务** | 后端 Java 调用地理编码等接口 |
| `农助手前端` | **Web端(JS API)** | 前端 H5 地图显示和定位 |

创建完成后会看到两个 Key，分别复制。

### 第四步：填入项目配置

**后端 Key** → 编辑项目根目录 `.env` 文件：
```
GAODE_MAP_API_KEY=后端Key粘贴到这里
```

**前端 Key** → 编辑 `frontend-uniapp/src/manifest.json` 中 H5 配置：
```json
{
  "h5": {
    "sdkConfigs": {
      "maps": {
        "amap": {
          "key": "前端Key粘贴到这里",
          "securityJsCode": ""
        }
      }
    }
  }
}
```

### 第五步：验证

**验证后端Key：** 浏览器打开（把 YOUR_KEY 换成后端Key）：
```
https://restapi.amap.com/v3/geocode/geo?address=河南省西平县盆尧镇&key=YOUR_KEY
```
返回 JSON 有 `location` 字段 = ✅

**验证前端Key：** 启动前端 `npm run dev:h5`，打开农户首页看地图是否正常加载 = ✅
