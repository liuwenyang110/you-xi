# 农助手公益平台 — 产品需求文档 (PRD)

**版本**: v3.0 公益重构版  
**日期**: 2026-04  
**定性**: 非营利性农业信息对接平台  

---

## 1. 产品定位

### 1.1 使命宣言

> **用技术服务农村，让农户与服务者自由连接。**  
> 平台完全免费，不收取任何中间费用，不参与任何商业交易，  
> 价格由农户与服务者双方直接面谈协商。

### 1.2 平台性质

| 维度 | 定义 |
|---|---|
| **性质** | 公益信息对接平台 |
| **盈利模式** | 无（非营利） |
| **核心功能** | 提供联系渠道，帮助农户找到有农机的人 |
| **不做什么** | 不接单、不撮合交易、不托管资金、不参与定价 |

### 1.3 目标用户

| 角色 | 定义 | 诉求 |
|---|---|---|
| **求助者（农户）** | 需要农机作业服务的农村家庭 | 快速找到本片区能帮忙的人 |
| **服务者（农机手）** | 拥有农机设备、有余力提供服务的村民 | 让附近有需要的人知道自己能帮忙 |
| **平台管理员** | 各县市农业推广人员 | 了解辖区供需状态，优化资源配置 |

---

## 2. 核心流程

```
农户发布求助需求
       ↓
需求公开展示于片区公告板
       ↓
服务者浏览需求 / 农户主动搜索服务者
       ↓
双方联系方式对接（平台记录联系行为，不参与内容）
       ↓
农户与服务者线下直接沟通、面谈价格
       ↓
服务完成后可选填写反馈评价（仅供参考，不强制）
```

**关键原则**：平台在"联系方式对接"后完全退出，后续交易与平台无关。

## 3. 功能模块

### 3.1 农户端（Uni-app）

#### 求助需求发布
- **入口**：首页底部发布按钮 → 选择"发布求助需求"
- **必填**: 作业类型、地块位置
- **选填**: 作业面积、期望时间、补充说明
- **发布后**: 需求立即出现在所在片区公告板

#### 服务者浏览与联系
- **服务者名片**: 姓名、拥有设备、服务范围、评价评分
- **联系方式**: 点击后展示（平台记录一次联系日志）
- **公益提示**: "以下联系方式由服务者授权展示，请直接联系，平台不参与"

#### 我的求助记录
- 查看已发布的需求状态（等待联系 / 已联系 / 已完成）
- 主动关闭需求（找到帮助后操作）
- 选填完成反馈与星级评价

### 3.2 服务者端（Uni-app）

#### 服务能力登记
- 填写拥有的农机设备、服务类型、服务范围片区
- 上传证件照片（选填，提升可信度）
- 联系电话（非必须公开，可选"私信联系"）

#### 片区需求浏览
- 按所在片区展示农户需求列表
- 筛选：服务类型 / 紧急程度 / 发布时间
- 一键联系发布者

### 3.3 Admin 后台（Vue PWA）

#### 公益运营看板
- **今日新增求助**: 今日发布需求数
- **今日对接成功**: 今日联系行为记录数
- **注册服务者**: 已登记的服务者总数
- **覆盖村镇**: 有活动记录的片区数

#### 用户管理
- 查看求助者 / 服务者列表
- 异常账户处理（骚扰举报响应）
- 服务者资质审核（可选）

#### 片区管理
- 创建/编辑服务区域
- 查看片区求助-对接数据
- 导出月度对接统计报告

## 4. 技术规范

### 4.1 技术栈

| 层次 | 技术 |
|---|---|
| 移动端 | Uni-app（Vue3 Composition API） |
| Admin 后台 | Vue 3 + Vite |
| 后端 | Spring Boot 3.x |
| 数据库 | MySQL 8.0 + Flyway 迁移 |
| 缓存 | Redis（统计快照） |
| 对象存储 | 阿里云 OSS（头像/证件） |

### 4.2 设计规范

| 元素 | 规范 |
|---|---|
| **主色** | `#2D7A4F`（公益绿） |
| **辅助色** | `#5BA88A`（浅公益绿） |
| **背景色** | `#F5F9F6` |
| **字体** | Noto Sans SC（适老友好） |
| **最小触控区域** | 44×44px（适老设计） |
| **基础字号** | 16px，老年模式 20px |

### 4.3 核心接口

| 接口 | 说明 |
|---|---|
| `POST /api/v3/demands` | 发布求助需求 |
| `GET /api/v3/zones/{id}/demands` | 片区需求列表 |
| `POST /api/v3/contact-reveal/log` | 记录联系行为 |
| `GET /api/v3/contact-reveal/stats/platform` | 平台公益统计 |

## 5. 公益原则声明

1. **完全免费**: 平台对所有用户永久免费，不收取任何服务费、信息费、佣金
2. **不参与定价**: 服务价格由双方自行协商，平台不建议、不干预、不记录价格
3. **数据最小化**: 仅收集服务对接必要的信息，农户联系方式经用户授权后才对外展示
4. **用户自主**: 用户随时可以删除账号、撤回需求、关闭联系方式展示
5. **开放透明**: 平台运营数据（对接次数、覆盖片区）对外公开，接受社会监督
