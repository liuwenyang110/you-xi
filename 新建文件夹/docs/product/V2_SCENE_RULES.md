# 农助手 V2 - 场景与业务规则定义 (由 PM 专家出具)

## 1. “适老关怀模式” 规范
- **语音识别入口**: 首页必须悬浮持久化的语音按钮（麦克风图标）。
- **字段要求**: 农机预约不需要农户填报复杂的亩数，默认使用历史数据或“看地出价”。
- **代办人制度**: 新增 `agent_id` 关联，允许外地手机号为本地老人下单，状态同步推送到双方微信。

## 2. “政府留痕与公益核验” 规则
- 农户端完成预约并确认联系意向后，平台冻结记录。
- 农机作业必须提供两项证据：1. GPS 地理围栏打卡（位于注册土地 1km 内）；2. 现场作业照片上传。
- 证据收集齐全后，系统后台自动归档至“数字乡村治理大屏”，并向当地农业局接口推送 `claim_status=SUBMITTED`。

## 3. “时空排班 (12306模式)” 预约规则
- 农机主将“整天”划分为 3 个槽位：上午 (06:00-11:00)、下午 (13:00-18:00)、夜场 (18:00以后)。
- 农产旺季最多提前 7 天开放槽位预约。
- 为了防止恶意占座，槽位锁定后 15 分钟内未确认联系则自动释放。
